Preface · How to read this primer

The companion page pipeline_mathematics.html records what the running pipeline implements in notation. This document records how to read those quantities once they exist: what is being measured, which independence assumptions typically fail silently, and why two similar percentages can answer different scientific questions.

The intended reader tolerates technical vocabulary when each term is unpacked. Where useful we pair abstraction with a micro-example (small numbers, pencil-and-paper arithmetic) and an analogy from ordinary measurement.

Analogy (throughout). Treat the pipeline as a compound instrument—telescope, sensor, compression. Each stage sharpens, distorts, or drops information. A headline accuracy figure is an image file at the end of the chain; it is not the scene itself.

E1 · Experimental units, clustering, and “who repeated whom?”

The pipeline surfaces paired rows and document-level summaries. A fragile silent assumption is exchangeability: treating every paired segment as independent evidence about model quality. That assumption rarely holds.

Nested structure. Documents \(i=1,\ldots,D\) contain varying counts \(n_i\) of ground-truth segments. Segments inside one memo share genre, voice, topic, and repetitive formulae. Model errors on neighbouring clauses correlate because representations carry context. That yields positive intra-document correlation of residuals.

Worked micro-example. Document A has 100 paired segments; Document B has 5. Pooling all 105 rows into one corpus accuracy lets A contribute twenty times more mass than B—not because A is “more important,” but because micropooling weights segments equally regardless of document.

Unit for inference. The quantity you attach uncertainty to should drive resampling: document-level bootstrap or hierarchical models if segments cluster; segment-level i.i.d. models only if dependence is negligible or modeled.

Analogy. Five hundred answers from fifty related households are not five hundred independent polls.

E2 · Multiple endpoints, multiple comparisons, exploratory dignity

The evaluation touches category agreement, framing agreement, conjunction (“both match”), pairing coverage, and sometimes alternate alignments (content_rus vs index). Each answers a different question; jointly they form a multiple-testing landscape. Under noise, scanning many summaries elevates the chance that at least one looks striking.

Exploration is legitimate; clarity lies in declaring epistemic status. Confirmatory claims (for example, p < 0.05 on a pre-specified primary endpoint) call for multiplicity control (Bonferroni is blunt; Benjamini–Hochberg FDR is common when outcomes correlate).

Worked micro-example. Ten independent tests at \(\alpha=0.05\) each yield roughly \(1-(1-\alpha)^{10}\approx 0.40\) probability that at least one rejects under global nulls—absent malice. Correlation among metrics modifies that arithmetic but does not remove the issue.

Analogy. Finding coins in one dresser drawer does not prove every drawer is full of treasure.

E3 · Raw accuracy vs agreement corrected for chance (Cohen’s \(\kappa\))

Accuracy divides agreements by evaluated pairs. Under heavy class imbalance, high accuracy can coexist with a model that chiefly predicts the mode. Cohen’s kappa compares observed agreement \(p_o\) to expected agreement \(p_e\) from fixed marginal prevalences:

\[ \kappa \;=\; \frac{p_o - p_e}{1 - p_e}. \]

\(\kappa=0\) matches chance given margins; \(\kappa=1\) is perfect agreement.

Worked micro-example (binary). Categories \(\{\mathrm{S},\mathrm{N}\}\). Independent chance baselines with \(\mathbb{P}(S)=0.9\) for both raters give \(p_e = 0.9^2 + 0.1^2 = 0.82\). Observed \(p_o=0.85\) yields \(\kappa=(0.85-0.82)/(1-0.82)\approx 0.167\)—high raw agreement, modest correction.

Pipeline hook. Parse-time coercion toward default taxonomy labels can concentrate margins artificially; inspect confusion matrices alongside \(\kappa\).

Analogy. Forecasting daily sunshine in a dry climate earns a high hit rate; \(\kappa\)-style reasoning asks what baseline deserved beating.

E4 · Alignment as latent correspondence and greedy approximation

Before label comparison, matching \(\mathcal{P}_i\) links human and predicted rows via \(\eta(\cdot)\)-normalized text. One may view this as estimating which phrases correspond when segmentation differs.

The implementation is deterministic greedy (exact match pass, then containment); it need not maximize global overlap or minimize edit distance—those would be different optimization problems.

Micro-example. If one predicted span could align with either of two GT spans, early greedy locking can block a globally better pairing.

Reported accuracy is therefore conditional on alignment-induced measurement error.

Analogy. Greedy seat assignment at an event is fast; it need not be globally optimal.

E5 · Prefix truncation \(\pi_k\) as irreversible bottleneck

The LLM prompt embeds \(\pi_{8000}(\mathrm{RUS})\)—see pipeline notation (L2 · truncation). Distinct full texts that agree up to character 8000 collide under \(\pi_k\); tail-only entities vanish from the model’s stimulus though ground truth may still reference them.

Tokenizer geometry matters: truncation is in Unicode characters before server-side BPE; semantic “attention budget” is not uniform across scripts.

Mental experiment. A decisive paragraph sits after the cutoff; GT encodes it; the model never sees it—errors need not reflect “ideology,” only missing input.

Analogy. Reading page one of a verdict skips the disposition on page twelve.

E6 · Requested JSON lines as a soft channel constraint

The prompt requests JSON lines; decoding is not grammar-constrained (see neural generator / L7.4 in the companion spec). \(\mathrm{Parse}\) keeps decodable lines and drops the rest, so invalid continuations erode structured rows and can trigger fallback.

You asked for a low-entropy subspace \(\mathcal{Y}_{\mathrm{json}}\subset \Sigma^\ast\); sampling still traverses \(\Sigma^\ast\) unless constrained decoding intervenes.

Analogy. Requesting haiku without counting syllables obtains occasional haiku-shaped text by cooperation, not by mechanical enforcement.

E7 · Stochastic decoding and Monte Carlo variability

For \(\tau>0\), continuations \(Y_i\) are random (see L7.3). Replicates differ unless sampling collapses and all seeds align across stacks. Treat \(\mathrm{Pred}_{\mathrm{LLM}}(i)\) as one draw, not a fixed functional of \(\mathrm{RUS}_i\).

Credible intervals over decode noise matter especially when \(n_i\) is small.

Tiny rehearsal. Validity of JSON may hinge on a delimiter realized probabilistically—accuracy can swing across rerolls with identical weights \(\theta\).

Analogy. One anemometer reading during a gust is data; several readings characterize turbulence.

E8 · Lineage: configuration graphs and deterministic forks

Reproducibility benefits from a DAG over artifacts: config, taxonomy, texts, weights, prompts, decode outputs, paired comparisons. Some edges are deterministic given parents (\(\eta\), greedy pairing at fixed inputs); decode is stochastic.

Fig E8 — Decode is the primary stochastic fork; reporting is deterministic given Pred/GT inputs.

Analogy. Same dough, different ovens—note where temperature stops being controlled.

E9 · Taxonomy as measurement instrument

Finite \(\mathcal{C}\) and \(\mathcal{F}\) operationalize qualitative theory (cf. Fig 2). The taxonomy is an instrument: it buckets continuous rhetorical nuance. Boundary disputes concentrate where humans also disagree—often where models flicker.

Changing taxonomy versions without relabelling breaks strict comparability across studies.

Analogy. Pain scales compress subjective experience; raw averages ignore anchor calibration differences.

E10 · Aggregation across documents

Pooled segment metrics weight long documents heavily unless reweighted. Macro averaging assigns equal weight per document; micro pooling weights each segment equally.

Contrast. Micro: \(\sum_i\sum_{r\in\mathcal{R}^{\mathrm{p}}_i}\mathrm{cat}(r) \,/\, \sum_i|\mathcal{R}^{\mathrm{p}}_i|\). Macro: \(\frac{1}{D}\sum_i \mathrm{Acc}^{\mathrm{cat}}_i\).

Analogy. GPA by course count differs from GPA by credit hour.

E11 · Blind versus anchored assessment protocols

Human workflows may be blind to gold labels or anchored to them (see repository agent handbook). Those protocols estimate different quantities; pooling without stratification mixes estimands.

Analogy. Blind taste tests and labelled taste tests measure different cognitive processes.

Caveat. Formal statements are conditional (“given greedy pairing,” “given \(\pi_{8000}\),” “given one decode draw”). Historical interpretation needs archival reasoning beyond any metric sheet.
